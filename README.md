# Projeto

## Como editar este código?

Você pode editar a aplicação localmente com sua IDE preferida.

Pré-requisitos: Node.js e npm instalados — consulte [nvm](https://github.com/nvm-sh/nvm#installing-and-updating).

Passos:

```sh
# 1) Clonar o repositório
git clone <YOUR_GIT_URL>

# 2) Entrar na pasta do projeto
cd <YOUR_PROJECT_NAME>

# 3) Instalar dependências
npm i

# 4) Iniciar o servidor de desenvolvimento
npm run dev
```

## Tecnologias

- Vite
- TypeScript
- React
- shadcn-ui
- Tailwind CSS

## Deploy

O processo de deploy depende do seu provedor (por exemplo, Netlify, Vercel, etc.). Gere o build com `npm run build` e publique a pasta de saída conforme a plataforma escolhida.
